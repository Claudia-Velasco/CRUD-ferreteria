# CRUD-ferreteria
Sistema: Ferreteria
<br/>
Nombre de la bd: ferreteria

Descripción: 
Cuenta con 3 tipos de usuarios: Administrador, empleado de ventas y bodega.
Cada usuario tiene función diferente, pero cada uno registra, modifica y elimina.
